JCF ALERT & RSS NEWS DASHBOARD

Real-time News Intelligence for India — Tool Specifications

⚡ V1  ·  Builder Not Coder

📋Overview

JCF Alert & RSS News Dashboard is a self-hosted Python/Flask web application that aggregates 191+ live RSS feeds from Indian news sources, auto-verifies articles with AI, plots city-level alerts on an interactive India map, and provides a full team workspace with role-based access, internal chat, and multi-format exports — all running from a single Python file.

⚙️Technical Specifications

🐍

Runtime

  • Python 3.9 – 3.12
  • Flask 2.3+
  • Single-file architecture
  • Windows · Linux · macOS
🗄️

Database

  • Primary: Microsoft SQL Server via pyodbc
  • Fallback: SQLite (zero setup)
  • Auto-detects & switches silently
  • WAL mode + threading locks
  • 30-day data retention (configurable)
📡

RSS Engine

  • 191+ India news feeds
  • Auto-fetch: 5 / 10 / 15 / 30 / 60 min
  • Manual fetch trigger in header
  • Custom feed manager in Settings
  • Duplicate detection via URL hash
🤖

AI Verification

  • Groq LLM API (optional key)
  • Heuristic fallback (15+ signals)
  • Results: ✅ Real / ❌ Fake / ⚠️ Unverified
  • Source credibility scoring
  • Confidence % per verdict
🗺️

Map & Geo

  • Leaflet.js interactive India map
  • City-level pin detection (500+ cities)
  • Pins coloured by category & risk
  • Click pin → local news panel
📤

Exports

  • Excel (.xlsx) — multi-sheet workbook
  • PDF — fpdf2 (auto-installs)
  • CSV — always available
  • Respects all active filters
🌐

Network

  • Binds 0.0.0.0:5012 (LAN accessible)
  • QR code for mobile access
  • Session-based authentication
  • LAN URL shown in Settings
📝

Logging

  • Daily: jcfalert_YYYYMMDD.log
  • 7-day auto-retention
  • Console + file dual output
  • Device login tracking per user
